# Fresh NextJs Installer

I install a lot of times the base nextjs, turborepo, shadcn and next forge base repositories. So I made a small utility to do it faster.

```sh
./init.sh
```

## Options

```sh
./init.sh "Next.js"
./init.sh "Turborepo"
./init.sh "shadcn Next"
./init.sh "shadcn Turbo"
./init.sh "Next Forge"
./init.sh "Install All"
```
